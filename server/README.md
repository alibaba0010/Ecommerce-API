## Quick Start

Add a
MONGO_URL
EMAIL_USER
JWT_SEC
JWT_LIFETIME
EMAIL_USER
EMAIL_PASS
EMAIL_HOST= .com
CLIENT_URL
CLOUD_NAME
CLOUD_API_KEY
CLOUD_API_SECRET
GEOCODER_API_KEY
GEOCODER_PROVIDER
to the ".env" file.

```bash
# Install dependencies
npm install

# Serve on localhost:5000
npm run dev (nodemon)

# Routes
GET    /api/v1/stores # Get Stores

POST   /api/v1/stores # Add Store

```
